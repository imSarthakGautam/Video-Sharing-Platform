# Video-Sharing-Platform

## Installing the dependencies.
```bash
npm i mongoose express dotenv cookie-parser cors bcrypt jsonwebtoken
npm i --save-dev nodemon
npm i mongoose-aggregate-paginate-v2
npm i cloudinary multer
```
